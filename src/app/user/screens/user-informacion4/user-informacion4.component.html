GitHub sufre una caída afectando a miles de desarrolladres de software

GitHub, el servicio de hosting de desarrollo más importante de Internet ha sufrido una caída. La web permanece inactiva para miles de desarrolladores desde hace alrededor de una hora, y los usuarios están experimentando distintos tipos de errores.

La caída se ha relajado, y muchas de las páginas de GitHub funcionan, pero las operaciones de GIT y muchas partes de la web están caídas. Según podemos ver en la página de estado de GitHub, la web está presentando algunos fallos para numerosos usuarios

Gráfico Estado Github

La totalidad del servicio no se ha visto afectada; el repositorio de código más popular entre desarrolladores está sufriendo errores puntuales para algunos usuarios e interrupciones de servicio. De nuevo, los errores más comunes consistgen en los inicios de sesión, pero el problema reside en el IDE, cuyo acceso para muchos desarrolladores es imposible.

GitHub ya ha asegurado que está investigando la causa del problema. Los principales fallos que han experimentado los servicios de la web han sido errores en la carga de aplicaciones. En estos momentos, GitHub asegura haber encontrado la fuente de estos errores y están trabajando en "su recuperación".

Esperamos los problemas se solucionen pronto, ya que nos encontramos dentro de los perjudicados.
